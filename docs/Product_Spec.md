# Product Spec

## 项目名称

PaperAgent：基于 Agentic RAG 的论文拆解与科研知识库助手

## 用户画像

- 计算机技术与软件工程专业研究生
- 需要进行文献调研、论文精读和综述写作的科研人员

## 核心问题

传统文献管理工具擅长保存文件和引用信息，但难以提取论文结构化知识，也难以支持跨论文的语义问答和长期知识沉淀。

## 核心需求

| 编号 | 需求 | 优先级 | 验收标准 |
|---|---|---:|---|
| PRD-01 | 上传并解析论文 | P0 | 支持 PDF/TXT/MD，非法格式有明确错误提示 |
| PRD-02 | 结构化拆解论文 | P0 | 输出标题、作者、摘要、关键词、研究问题、方法、贡献、局限等字段 |
| PRD-03 | 本地知识库管理 | P0 | 论文和知识块可持久化保存、查询和删除 |
| PRD-04 | RAG 问答 | P0 | 能检索相关片段并生成带来源的回答 |
| PRD-05 | 会话与长期记忆 | P1 | 能保存多轮对话、会话标题和关键记忆 |
| PRD-06 | 分类管理 | P1 | 支持按分类管理论文并过滤检索范围 |
| PRD-07 | 批量导入 | P1 | 支持批量文件和目录导入 |
